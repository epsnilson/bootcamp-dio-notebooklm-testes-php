# php-software-testing-guide

## Como utilizar o NotebookLM para aprender Testes de Software com PHP

## Contexto

Este projeto foi desenvolvido como parte do Bootcamp da DIO com o objetivo de explorar o **NotebookLM** como ferramenta de aprendizagem ativa.

Para isso, escolhi estudar **Testes de Software com PHP**, dividindo o tema em duas etapas:

* **Parte 1:** Testes em PHP sem framework, utilizando PHPUnit.
* **Parte 2:** Testes em aplicações desenvolvidas com Laravel.

A proposta deste projeto não é criar um curso sobre testes de software, mas documentar o processo de aprendizagem utilizando o NotebookLM. Durante o estudo, foram selecionadas fontes confiáveis, elaborados diferentes prompts, refinadas perguntas e consolidados os principais conceitos aprendidos.

Ao final, este repositório reúne um pequeno guia de estudos, um glossário e um conjunto de prompts reutilizáveis que poderão servir como apoio em futuras revisões sobre o tema.

---

## Objetivos

### Objetivo Geral

Utilizar o NotebookLM como ferramenta de aprendizagem para compreender os principais conceitos de Testes de Software com PHP e registrar o processo de estudo realizado durante o Bootcamp.

### Objetivos Específicos

* Explorar o NotebookLM como apoio à organização e compreensão de conteúdos técnicos.
* Estudar os fundamentos dos testes de software aplicados ao desenvolvimento em PHP.
* Conhecer os principais recursos do PHPUnit para criação de testes automatizados.
* Compreender como o Laravel organiza e facilita a implementação de testes.
* Registrar a evolução dos prompts utilizados durante o estudo.
* Consolidar o aprendizado por meio de um miniguia, um glossário e uma coleção de prompts reutilizáveis.

---

## Estrutura do Projeto

O desenvolvimento deste estudo foi organizado nas seguintes etapas:

1. Seleção de fontes confiáveis.
2. Importação das fontes para o NotebookLM.
3. Formulação e refinamento de prompts.
4. Análise das respostas obtidas.
5. Consolidação do conhecimento em um miniguia de estudo.
6. Elaboração deste README como registro da experiência de aprendizagem.
